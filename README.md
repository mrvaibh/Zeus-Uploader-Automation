# Zeus-Uploader-Automation

## Setup
### For Development
 - `pip install pipenv`
 - `pipenv shell`
 - `pipenv install`
### For Production
 - `pip install -r requirements.txt`

## Usage
### Setting Up
 - run `setup.py`
 - enter SERVER NAME, MAX RETRIES, TIME DELAY, SCHEDULED TIME and MACHINE INFO
### Set on startup
 - Copy `startup.py`
 - press `Win+R`, type `shell:startup`
 - press `Ctrl+Shift+V`

## Troubleshoot
Delete the `logs.csv` file and try again in case of anonymous errors.
