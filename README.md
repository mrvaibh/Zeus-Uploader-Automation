# Zeus-Uploader-Automation

### Setup
 - `pip install pipenv`
 - `pipenv shell`
 - `pipenv install`

### Usage
 - create a file `machine_list`
 - add IP and machine ID per line
 example: ```
 192.168.88.201,11
192.168.88.201,11
192.168.88.201,11
192.168.88.201,11
192.168.88.201,11
```

### Cron
 - Make use if `script.bat` to setup CRON
